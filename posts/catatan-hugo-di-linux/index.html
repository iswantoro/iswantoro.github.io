Catatan Hugo Di Linux

Nov 29, 2021
1 min read

Ini adalah catatan pribadi saya, biar ketika membuat content baru di hugo tidak lupa cara post nya. hehehe

  1. buat post baru 

    $ hugo new posts/judul.md
    setelah selesai edit edit 
    $ hugo server

  2. kirim ke github pindah ke folder public dulu sebelum kirim ke github 

    $ cd public
$ git add .
$ git commit -m "post baru des"
$ git push origin main
$ git push -u origin master
